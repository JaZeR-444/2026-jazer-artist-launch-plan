# Module 05: Release Strategy Benchmarks

## Purpose

This module analyzes **release patterns, timing strategies, and momentum tactics** used by successful artists in JaZeR's niche. It defines optimal cadence, format (singles vs EPs vs albums), and pre-release positioning.

**Key Focus Areas:**
- Release frequency analysis of comparable artists
- Single rollout strategies (lead times, teaser content, pre-save campaigns)
- EP vs album performance data in the Melodic Rap space
- Seasonal timing and market windows (avoiding major releases, capitalizing on trends)
- Metadata strategy (featuring credits, producer tags, genre classification)

This module produces a templated release calendar with milestones, submission deadlines, and content requirements.

## Folder Structure & File Placement

This directory follows a file-type–based organization model so every asset has a clear home.

├── 01 - Research Prompts
├── 02 - Markdown Notes
├── 03 - Docx Research Reports
├── 04 - Data & Exports
├── 05 - Visuals & Screenshots
├── 06 - Slides & Presentations
├── 07 - References & Links
└── 08 - Final Outputs

## Folder Definitions

**01 – Research Prompts**
LLM prompts, research instructions, master queries, and exploration prompts used to generate or guide analysis.

**02 – Markdown Notes**
Working notes, drafts, iterative thinking, and structured markdown research files.

**03 – Docx Research Reports**
Compiled narrative documents intended for linear reading, review, or sharing.

**04 – Data & Exports**
Raw and processed data, CSV exports, metrics pulls, and third-party tool outputs.

**05 – Visuals & Screenshots**
Screenshots, diagrams, charts, UI captures, and image-based research artifacts.

**06 – Slides & Presentations**
Presentation decks, summary slides, and visual storytelling materials.

**07 – References & Links**
Source material, saved articles, PDFs, benchmark documentation, and external references.

**08 – Final Outputs**
Polished conclusions, summaries, and assets considered “decision-ready” or shareable.

## Usage Guidelines

*   This folder is topic-scoped, not time-scoped.
*   Drafts and incomplete work are expected.
*   Final conclusions should be promoted into **08 - Final Outputs** when ready.
*   File names should be descriptive and sortable, but perfection is not required.

## Relationship to the Larger Research System

This directory is one component of a 10-part Spotify research system, each module addressing a distinct aspect of artist setup, growth, or monetization.

Consistency in structure across all directories enables:
*   Faster navigation
*   Easier synthesis
