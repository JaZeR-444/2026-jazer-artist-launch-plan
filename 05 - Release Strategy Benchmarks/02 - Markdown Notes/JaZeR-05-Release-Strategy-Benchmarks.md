# 05 - Release Strategy Benchmarks: JaZeR

## 1. Strategy Benchmark Comparison

| Strategy | Pros | Cons | Verdict for JaZeR |
| :--- | :--- | :--- | :--- |
| **The "Album Drop"** | *Artistic Statement* | *Algorithm hates it (1 spike then death)* | *AVOID until established.* |
| **The "Waterfall"** | *Constant growth; each song boosts the last* | *High asset demand* | **BEST OPTION.** |
| **The "EP Tease"** | *Middle ground; builds a world* | *Slower than pure singles* | *Use for Phase 2.* |

## 2. Recommended Release Timeline (The "Waterfall Blueprint")

**Goal:** Release a new single every **4-6 weeks**. This keeps JaZeR in the "Release Radar" loop constantly.

*   **Month 1: Single 1 ("The Introduction")**
    *   *Vibe:* Accessibility. The most catchy, melodic track.
    *   *Tactic:* Pitch to "Fresh Finds."
*   **Month 2: Single 2 ("The Vibe")**
    *   *Vibe:* Deeper cut, more "Artistic/Vibe" focused.
    *   *Tactic:* Waterfall Single 1 onto this release (Single 2 is Track 1, Single 1 is Track 2). This gives Single 1 a "second life" of streams.
*   **Month 3: Single 3 ("The Banger")**
    *   *Vibe:* High energy or collaboration (e.g., with a local Austin artist).
    *   *Tactic:* Waterfall Single 2 + 1.
*   **Month 4: The EP (Compilation)**
    *   *Content:* The 3 singles + 2-3 new tracks.
    *   *Event:* "EP Release Party" at Empire Control Room.

## 3. Asset Checklist (The "Deliverables")

For *each* single, you need:

1.  **Cover Art:** High-quality, distinct style. (Consistency is key).
2.  **Spotify Canvas:** 3-8 second looping video. (CRITICAL for engagement).
3.  **Vertical Video (x3):**
    *   *Teaser:* "New sound coming..."
    *   *Launch:* "Out Now!" lip-sync/performance.
    *   *Story:* "How I made this beat."
4.  **Pre-Save Link:** Use a tool like Feature.fm or DistroKid.
5.  **Pitch Text:** 500-character elevator pitch for Spotify Editors.
